# Assets Directory

This directory contains static assets for the Soaps MythicMobs Editor.

## Current Files

- `favicon.svg` - Main favicon in SVG format (works in modern browsers)

## Required Image Assets (To Be Created)

These image files need to be created manually using an image editor:

### 1. `og-image.png` (1200x630 pixels)
Open Graph image for social media sharing. Should include:
- Site logo/dragon icon
- "Soaps MythicMobs Editor" title text
- Brief tagline like "Create Custom Mobs & Skills"
- Brand colors: Purple (#8b5cf6), Blue (#3b82f6), Dark (#1a1a2e)

### 2. `favicon-32.png` (32x32 pixels)  
PNG favicon for older browsers that don't support SVG favicons.
Can be exported from the favicon.svg file.

### 3. `apple-touch-icon.png` (180x180 pixels)
Apple Touch Icon for iOS devices.
Should be the dragon icon on a solid purple background.
